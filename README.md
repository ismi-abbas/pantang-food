# Pantang Food

Pantang Food is a TanStack Start app for Malay moms to browse, save, and extend postpartum-friendly recipes.

## Stack
- TanStack Start
- React 19
- Tailwind CSS v4
- Drizzle ORM
- Turso / libSQL
- Cloudflare Vite plugin

## Local development

```bash
cp .env.example .env
pnpm install
pnpm dev
```

Default local database:
- `file:./local.db`

## Useful commands

```bash
pnpm test
pnpm build
pnpm lint
pnpm db:generate
pnpm db:push
```
